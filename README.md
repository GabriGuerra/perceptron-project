Experimental study using a Multilayer Perceptron (MLP) applied to a real dataset, with emphasis on generalization behavior, overfitting, and underfitting.

## Tech Stack
- Python
- scikit-learn
- NumPy
- Jupyter Notebook

## How to run
```bash
git clone https://github.com/GabriGuerra/mlp-perceptron-study.git
cd mlp-perceptron-study
pip install -r requirements.txt
jupyter notebook
```
