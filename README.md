# perceptron-project



Implementation and experiments with the Perceptron algorithm on synthetic and real datasets.

## Tech Stack
- Python
- scikit-learn
- Jupyter Notebook

## How to run
```bash
git clone https://github.com/GabriGuerra/perceptron-project.git
cd perceptron-project
pip install -r requirements.txt
jupyter notebook perceptron.ipynb
```
